  • Ortalaşdırılmayan Xərclər: Ailə və dost paketlərində ortaq xərclərin ədalətli bölünməməsi.
  • Limitsiz Kart Təhlükəsi: Kənar xidmətlərə verilən kart nömrələrinin (PAN) kiber-təhlükə altında qalması.

02 / HƏLL

Abunəm ile Mərkəzi Nəzarət

Abunəm bütün abunəlik xərclərini, limitlərini və ödəniş metodlarını vahid paneldən tam nəzarətdə saxlamağa imkan verir:

  • Gecikməyən Xəbərdarlıqlar: Ödənişə 1-3 gün qalmış abunəçiyə avtomatik bildirişlər göndərilir.
  • Aylıq Büdcə Limiti: Limit aşılarsa, sistem sizi dərhal məlumatlandırır və yeni xərcləri tənzimləyir.
  • Ortaq İstifadə (Split Billing): Dostlarınızla abunəliklər yaradıb ödəniş faizlərini ədalətli şəkildə bölüşün.

03 / ARXİTEKTURA

C4 Model — Sistem Diaqramı

Sistemimiz tamamilə genişlənə bilən, PCI-DSS standartlarına uyğun 3 layerli struktura malikdir:

  • L1: System Context: İstifadəçi, Abunəm platforması və kənar inteqrasiyaların (Email, Ödəniş Gateway) əlaqəsi.
  • L2: Containers: Veb-tətbiq interfeysi, REST/Swagger API serveri və PostgreSQL verilənlər bazası.
  • L3: Components: Auth, Subscription, Payment, Settings və Notification menecerləri.
04 / TƏHLÜKƏSİZLİK

Yüksək Kiber-Təhlükəsizlik Qoruması

Kart Nömrələrinin Maskalanması
Müştərilərin 16 rəqəmli kart nömrələri (PAN) heç vaxt bazaya birbaşa yazılmır. Bazada və API cavablarında kartlar yalnız PCI-DSS uyğun maskalanmış formatda saxlanılır.
Helmet Təhlükəsizlik Headers
Express serverimizdə Helmet modulu aktivdir. Bu başlıqlar vasitəsilə Clickjacking, MIME Sniffing və XSS (Cross-Site Scripting) kimi ümumi kiber hücumların qarşısı tamamilə alınır.
Bruteforce & DDoS Qoruması
Giriş və qeydiyyat API marşrutlarında Rate Limiter aktivləşdirilmişdir. Eyni IP-dən 15 dəqiqədə 30-dan çox sorğu göndərilərsə, sistem həmin IP ünvanı avtomatik olaraq müvəqqəti bloklayır.
SQL Injection Qoruması
Bütün verilənlər bazası sorğuları tam olaraq parametrləşdirilmiş (SQL parameterization) formatda yazılmışdır. API üzərindən kənar SQL komandalarının göndərilməsi qeyri-mümkündür.
05 / GƏLƏCƏK

Layihənin Yol Xəritəsi (Roadmap)

Abunəm platformasının növbəti mərhələlərdə genişləndirilməsi planları:

1
Open Bank İnteqrasiyası
Yerli və xarici bankların API-ləri vasitəsilə kart xərclərinin avtomatik sinxronizasiyası.
2
Ödəniş Gateway İnteqrasiyası
Birbaşa tətbiq daxilindən abunəlik ödənişlərini həyata keçirmək üçün ödəniş sistemlərinin qoşulması.
3
Mobil Tətbiqlər (iOS / Android)
Flutter və ya React Native vasitəsilə yerli mobil tətbiqlərin App Store və Play Store-da buraxılması.
