Sistem Konteksti (L1 Context)

Abunəm platformasının xarici sistemlər və istifadəçilərlə qarşılıqlı əlaqə sərhədləri.

Məqsəd

Sistemin biznes mühitində necə yerləşdiyini, istifadəçilərin kim olduğunu və xarici inteqrasiya nöqtələrini yüksək səviyyədə göstərmək.

Əsas Elementlər

İstifadəçi (Abunəçi) Person
Platformaya daxil olaraq öz abunəliklərini, büdcəsini və kartlarını idarə edən müştəri.
Abunəm Platforması System
İstifadəçilərin abunə xərclərinə nəzarət edən, büdcə hesablayan əsas proqram təminatı.
Ödəniş Provayderi Ext. System
Kart balanslarının yoxlanılması və ödəniş əməliyyatlarının aparılmasını təqlid edən mock bank API-ı.
SMTP Mail Server Ext. System
Yaxınlaşan ödənişlər və büdcə limitinin keçilməsi haqqında istifadəçiyə e-poçt göndərən xarici SMTP xidməti.
